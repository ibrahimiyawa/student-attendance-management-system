# Student Attendance Management System

## Overview
This is a web-based attendance management system developed using PHP and MySQL.  
It allows teachers and administrators to manage student attendance, authentication, and generate reports.

## Features
- User login & authentication
- Teacher and admin roles
- Attendance marking
- Attendance history
- Report generation

## Technologies Used
- PHP
- MySQL
- HTML/CSS
- JavaScript
- XAMPP

## Installation
1. Install XAMPP
2. Start Apache and MySQL
3. Import `attendance_db.sql` into phpMyAdmin
4. Copy project folder into `htdocs`
5. Visit:
http://localhost/AttendanceSystem
